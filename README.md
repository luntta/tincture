# Tincture

A multi purpose color tool. Work in progress.

Usage:
```
tincture("rgb(0,0,255)");
tincture("rgb(0 0 255)");
tincture("rgb(0%,0%,100%)");
tincture("rgb(0% 0% 100%)");
tincture("rgba(0,0,255,0.5)");
tincture("rgba(0 0 255 / 0.5)");
tincture("rgba(0%,50%,100%,0.5)");
tincture("rgba(0%,50%,100%,50%)");
tincture("rgba(0% 50% 100% / 0.5)");
tincture("rgba(0% 50% 100% / 50%)");
tincture("#0000ff");
tincture("#00f");
tincture("#0000ff80");
tincture("#00f8");
tincture("hsl(240,100%,50%)");
tincture("hsl(240deg,100%,50%)");
tincture("hsl(3.14rad,100%,50%)");
tincture("hsl(0.5turn,100%,50%)");
tincture("hsl(240 100% 50%)");
tincture("hsl(240deg 100% 50%)");
tincture("hsl(3.14rad 100% 50%)");
tincture("hsl(0.5turn 100% 50%)");
tincture("hsla(240,100%,50%,0.5)");
tincture("hsla(240deg,100%,50%,0.5)");
tincture("hsla(3.14rad,100%,50%,0.5)");
tincture("hsla(0.5turn,100%,50%,0.5)");
tincture("hsla(240,100%,50%,50%)");
tincture("hsla(240deg,100%,50%,50%)");
tincture("hsla(3.14rad,100%,50%,50%)");
tincture("hsla(0.5turn,100%,50%,50%)");
tincture("hsla(240 100% 50% / 0.5)");
tincture("hsla(240deg 100% 50% / 0.5)");
tincture("hsla(3.14rad 100% 50% / 0.5)");
tincture("hsla(0.5turn 100% 50% / 0.5)");
tincture("hsla(240 100% 50% / 50%)");
tincture("hsla(240deg 100% 50% / 50%)");
tincture("hsla(3.14rad 100% 50% / 50%)");
tincture("hsla(0.5turn 100% 50% / 50%)");
tincture("blue");
```
